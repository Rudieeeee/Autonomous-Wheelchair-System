# Autonomous Wheelchair System

## Overview
This project focuses on the development of an **autonomous smart wheelchair** designed to improve mobility and independence for users with limited motor control. The system operates in a **known indoor environment**, where a user selects a destination point, after which the wheelchair autonomously navigates to that location.

To achieve this, the system combines **localization, path planning, and obstacle avoidance** to ensure safe and reliable navigation. The architecture is designed to be modular, allowing for integration of multiple input methods such as a graphical interface, eye-tracking, or EMG-based control.

---

## Key Features
- 🧭 Autonomous navigation to a user-selected destination  
- 📍 Real-time localization within a known indoor map  
- 🚧 Obstacle detection and collision avoidance  
- 🧑‍🦽 Multiple user input methods (GUI, eye-tracking, EMG, manual control)  
- 🔌 Modular and extensible system design  

---

## System Architecture
The system is divided into four main subsystems:

1. **User Input Subsystem**  
   Handles user interaction and destination selection.

2. **Positioning Subsystem**  
   Responsible for map generation and localization of the wheelchair.

3. **Navigation Subsystem**  
   Computes and executes safe, collision-free paths.

4. **Integration Subsystem**  
   Manages communication, power, and coordination between all components.
