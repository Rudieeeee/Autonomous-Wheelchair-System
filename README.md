# Autonomous-Wheelchair-System
An autonomous smart wheelchair system that enables hands free navigation using eye tracking and EMG signals. The project integrates biosignal acquisition, depth estimation, and real time pathfinding to allow users, particularly quadriplegic patients, to safely navigate their environment or autonomously reach selected destinations.
